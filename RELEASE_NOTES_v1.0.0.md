# Release v1.0.0 - 第一个版本

## 🎉 首个正式版本发布

纲一下的第一个正式版本，一个优雅的桌面应用，用于随机播放郭德纲相声。

## ✨ 主要功能

### 🎵 核心播放功能
- ✅ 随机播放郭德纲相声
- ✅ 单口/对口分类搜索
- ✅ 播放列表管理
- ✅ 多种播放模式（顺序、循环、单曲、自动）

### 🎛️ 播放控制
- ✅ 播放速度调节（0.5x - 2.0x）
- ✅ 音量控制和静音切换
- ✅ 进度条拖拽控制
- ✅ 上一首/下一首切换

### ❤️ 收藏功能
- ✅ 一键收藏喜欢的相声
- ✅ 收藏列表管理
- ✅ 从收藏列表快速播放

### 💾 下载功能
- ✅ 音频文件下载
- ✅ 下载进度显示
- ✅ 自定义下载路径

### 🎨 界面特性
- ✅ 现代化深色主题 UI
- ✅ 音频波形可视化
- ✅ 系统托盘支持
- ✅ 无边框自定义窗口

## 📦 技术栈

- React 18 + TypeScript
- Electron 28
- Vite 5
- Zustand
- Tailwind CSS

## 📥 下载

请从 [Releases](https://github.com/flystar233/Gang/releases) 页面下载对应平台的安装包。

## 🚀 使用说明

1. 下载并安装应用
2. 点击 "Gang单口" 或 "Gang对口" 开始播放
3. 使用播放控制按钮管理播放
4. 点击心形图标收藏喜欢的相声

## 📝 注意事项

- 需要网络连接才能获取相声资源
- 首次使用可能需要等待资源加载
- 建议保持应用更新以获得最佳体验
